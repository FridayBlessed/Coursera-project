## Peer-graded Assignment: Using Microsoft Copilot for Writing HTML, CSS, and JavaScript
---------------------------------------------------------------------

### Grading Criteria Overview

There are a total of 20 points for this project. 

- (5pts) Did you use HTML to create a structure for your portfolio website? 

- (3pts) Does your HTML code include accessibility features like alt tags? 

- (5pts) Did you use CSS to apply a style across your portfolio website? 

- (5pts) Did you use JavaScript to add interactivity to your portfolio website?   

- (2pts) Does your CSS ensure that your website is responsive across different screen sizes?
